\# Dataset



O dataset utilizado neste projeto não está incluído no repositório por questões de tamanho e licença.



\## Download



Você pode baixar o dataset no Kaggle:



https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews



\## Instruções



1\. Faça o download do arquivo

2\. Extraia o conteúdo

3\. Coloque o arquivo `IMDB Dataset.csv` nesta pasta (`data/`)



\## Estrutura esperada



data/

└── IMDB Dataset.csv
