📊 ANÁLISE DE SENTIMENTOS COM MACHINE LEARNING

🧠 Objetivo  
Classificar textos (reviews) como positivos ou negativos utilizando Machine Learning.

⚙️ Tecnologias utilizadas  
Python, Pandas, NumPy, Scikit-learn, TF-IDF e Regressão Logística.

🧩 Como funciona  
O projeto utiliza um pipeline com:
- Vetorização de texto com TF-IDF  
- Modelo de Regressão Logística para previsão de sentimentos
