# analise-sentimentos-ml
Projeto de Machine Learning para análise de sentimentos em textos utilizando NLP.  O modelo utiliza TF-IDF e Regressão Logística para classificar avaliações como positivas ou negativas.  Inclui pipeline completo de treinamento, avaliação e predição, com possibilidade de expansão para uso em atendimento ao cliente (CSAT/NPS).
